## aタグのリンクを新しいタブで開く方法
以下のように属性を設定すると実現できる。
```
<a href="[リンク先URL]" target="_blank">リンク</a>
```
## target="_blank"のセキュリティ問題
実は**target="_blank"で別タブで開いたリンク先とリンク元は同一のページとして扱われてしまう。**  
要は**リンク先からリンク元をいじることができてしまう**ということ。
