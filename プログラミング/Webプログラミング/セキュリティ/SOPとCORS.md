# SOPとCORS

## そもそもオリジンとは？
**オリジン = スキーム + ホスト + ポート**

例) `http://www.example.com:80/index.html`のURLを例にすると
- スキーム = `http://`
- ホスト = `www.example.com`
- ポート = `80`(ポート番号「80」は省略可なので普段見かけることは少ない)

⇒ オリジン = `http://www.example.com:80`
