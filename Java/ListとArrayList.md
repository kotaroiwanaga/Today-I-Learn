# コレクション
配列のように多数の値をまとめて操作でき、可変長であるもの。

#### リスト(List)　
順序通りに並べて格納

#### セット(Set)　
順序があるとは限らず格納

#### マップ(Map)　
ペアで対応づけて格納  


# ListとArrayList

### List
リストにあたるクラス型であり、インターフェース。

### ArrayList
Listを実装したクラスの一つ(Listの子クラス)。  
配列(Array)のような感覚で使用できる。  

### 使い分け
ほかのList型のクラス(LinkedListなど)と併用するならList、  
ArrayList独自のメソッドを使いたいならArrayList  
(あとでちゃんと調べたほうがいいかも)
