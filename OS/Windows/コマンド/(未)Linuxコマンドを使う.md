### 方法2:busyboxコマンド

- 下記よりbusybox.exeをダウンロード
http://frippery.org/busybox/

- ダウンロードしたbusybox.exeのディレクトリの環境パスを通す

- 以下のコマンドを実行するとシェルが起動し、Linuxコマンドが使えるようになる
```
> busybox sh -l
```

やめるときはexit
```
> exit
```
