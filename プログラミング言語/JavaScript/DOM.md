## DOMとは
「Document Object Model」の略で、Webページ(HTML)と他の言語(JavaScriptなど)を結び付けてくれる仕組み。 
HTMLのタグ構造を階層構造になったオブジェクトとして扱う。  
具体的に言うとJavaScriptでいう「document.~」て書くやつはDOMの仕組みを使ってる。
